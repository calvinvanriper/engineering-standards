## [1.0.0] - Initial baseline established

Initial engineering standards release including:

- Repository structure standards
- Git workflow and commit discipline
- Documentation standards
- Naming conventions
- README and CONTRIBUTING templates
- Issue and PR templates
- ESLint, Prettier, and VS Code baseline configs
- Adoption guide
- ADR governance system

This version establishes the baseline engineering contract for all adopting repositories.
